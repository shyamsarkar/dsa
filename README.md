# 🧠 Ruby Interview Tips

- Prefer clarity over cleverness
- Use:
  - `Hash.new(0)`
  - `each_with_index`
  - `Array#push`, `Array#pop`
- Avoid heavy Ruby magic in interviews
- Always explain your approach before coding
